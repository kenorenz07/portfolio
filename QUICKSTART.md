# Keno Renz Bacunawa - Portfolio

Personal portfolio website showcasing my work, skills, and experience as a Full Stack Web & Mobile Developer.

## Quick Start

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000)

## Features

✅ Modern Next.js 14 with App Router
✅ TypeScript for type safety
✅ Tailwind CSS for styling
✅ Framer Motion animations
✅ Dark mode support
✅ Fully responsive design
✅ SEO optimized
✅ Vercel deployment ready

## Deployment

Simply push to GitHub and import in Vercel for automatic deployment.

For detailed information, see [README.md](README.md)
